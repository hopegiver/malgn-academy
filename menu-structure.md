# 학원관리솔루션 메뉴 구성도

> 직관적이고 쉬운 학원 운영을 위한 핵심 메뉴 구조

---

## 📋 전체 메뉴 구조 개요

### 💎 상품 등급 안내

**[S] Standard** - 소형 학원 (재원생 100명 이하)
- 기본적인 학원 운영에 필요한 핵심 기능
- 단일 지점 운영
- 월 구독료: 기본 요금

**[P] Professional** - 중형 학원 (재원생 100~300명)
- Standard 기능 + 자동화 및 고급 기능
- 결제톡, 카드단말기 연동, 출결 앱 등
- 월 구독료: Standard + 추가 요금

**[E] Enterprise** - 대형/다지점 학원 (재원생 300명 이상, 또는 2개 이상 지점)
- Professional 기능 + 다지점 관리 및 고급 분석
- 지점 관리, 통합 분석, ERP 연동 등
- 월 구독료: Professional + 추가 요금 (또는 별도 견적)

### 네비게이션 UI 구성

**상단 GNB (Global Navigation Bar)**
- 1depth 메뉴 가로 배치 또는 상단 고정
- 좌측: 로고, 학원명
- 중앙: 1depth 메뉴
- 우측: **🔔 알림 센터**, 사용자 정보, 로그아웃

**🔔 알림 센터** (모든 상품)
- GNB 우측 상단에 벨 아이콘 배치
- 읽지 않은 알림 개수 배지 표시
- 클릭 시 드롭다운으로 최근 알림 표시
- **주요 알림 유형**:
  - 💰 미수납 학생 알림 (납부 기한 임박/초과)
  - 📅 상담 예약 알림 (오늘/내일 예정)
  - 📚 수강 만료 예정 알림 (D-7, D-3, D-1)
  - 🎂 생일 알림 (학생/강사 생일)
  - 📝 결석 연속 알림 (3일 이상 연속 결석)
  - 👨‍🏫 강사 휴가 알림
  - 🏆 목표 달성 알림 (월 매출 목표 등)
- **알림 액션**:
  - 각 알림에 바로가기 버튼 (해당 화면으로 즉시 이동)
  - 읽음/삭제/전체 읽음 처리
  - 알림 설정 (알림 ON/OFF, 알림 시간 설정)

**좌측 사이드바 (LNB - Local Navigation Bar)**
- 선택한 1depth의 2depth 메뉴 표시
- 심플한 리스트 형태

**페이지 영역**
- 실제 컨텐츠 표시
- 탭/모달로 세부 기능 처리

## 👥 학생관리

**라우트**: `/students` (1depth)

### 2depth 메뉴 구성
1. 학생 목록
2. 학생 등록
3. 상담 관리

---

### 1. 학생 목록
**라우트**: `/students/list`

**주요 기능**
- 전체 학생 조회 (재원생/퇴원생 탭)
- 통합 검색창 (이름, 연락처, 학교)
- 필터 (학년, 수강 상태, 등록일)
- 정렬 (이름, 등록일, 학년)
- 엑셀 다운로드/업로드

**표시 정보**
- 사진, 이름, 학년, 학교
- 연락처, 등록일
- 현재 수강 강좌
- 미수납 여부 (빨간색 표시)
- **빠른 액션**: [전화] [문자] [상세] [수납] [수강신청]
  - 전화: 클릭 시 바로 전화 걸기 (모바일)
  - 문자: SMS 발송 화면으로 바로 이동
  - 상세: 학생 상세 페이지
  - 수납: 수납 처리 화면으로 바로 이동 (학생 자동 선택)
  - 수강신청: 수강 신청 화면으로 바로 이동 (학생 자동 선택)

**학생 상세 페이지** (클릭 시)
- 라우트: `/students/detail?id={studentId}`
- 탭 구성:
  - 기본정보: 수정 가능
  - 수강내역: 현재/이전 수강 목록
  - 수납내역: 납부/미납 내역
  - 출결현황: 출석률, 출결 캘린더
  - 성적관리: 성적 목록, 추이 그래프
  - 상담일지: 상담 기록

### 2. 학생 등록
**라우트**: `/students/register`

**입력 폼**
- 기본정보: 이름, 생년월일, 성별
- 학교정보: 학교명, 학년
- 연락처: 학생 연락처, 학부모 연락처 (2개)
- 주소: 우편번호 검색
- 특이사항: 알레르기, 건강상태 등
- 사진 업로드

**하단 버튼**
- [저장] [저장 후 수강신청] [취소]

### 3. 상담 관리
**라우트**: `/students/counseling`

**주요 기능**
- **상담 일정 관리**
  - 상담 일정 캘린더
  - 상담 예약 등록
  - 상담 완료/미완료 처리

- **상담 요청 관리**
  - 학부모 상담 요청 접수
  - 웹/앱에서 받은 상담 요청
  - 요청 상태 (대기/확정/완료/취소)
  - 상담 일정 자동 배정
  - 상담 확정 알림 발송

- **상담 기록**
  - **유형/주제별 상담 분류**
    - 입학 상담
    - 학습 상담
    - 진로 상담
    - 생활 상담
    - 성적 상담
  - 상담 내용 상세 기록
  - 상담 결과 및 조치사항
  - 다음 상담 예정일
  - 첨부파일 (상담 자료)

- **상담 현황**
  - 학생별 상담 이력 조회
  - 상담 유형별 통계
  - 상담 완료율
  - 상담 만족도

**중대형 학원을 위한 고급 기능**
- **학생 일괄 관리**
  - 학년 일괄 업데이트 (학년 자동 진급)
  - 선택 학생 일괄 SMS/알림톡 발송
  - 일괄 수업 배정/해지
  - 엑셀 일괄 등록/수정
- **학생 그룹 관리**
  - 그룹 생성 (반, 팀, 버스노선 등)
  - 그룹별 관리 및 통계
  - 그룹별 메시지 발송
- **형제/자매 연결 관리**
  - 가족 단위 관리
  - 형제 할인 자동 적용
  - 가족별 통합 수납
- **졸업생 관리**
  - 졸업생 데이터 보관
  - 졸업생 재등록 프로세스
  - 졸업생 통계
- **고급 검색 및 필터**
  - 복합 조건 검색 (나이, 주소, 등록일 등)
  - 저장된 검색 조건
  - 맞춤 필터 생성

### 1. 수업 관리
**라우트**: `/classes/manage`

**주요 기능**
- 수업 목록 조회
- 새 수업 개설 (모달 또는 우측 패널)
- 수업 수정/삭제
- 검색/필터 (과목, 강사, 요일, 시간)

**수업 목록 표시**
- 수업명, 과목, 담당 강사
- 요일/시간, 강의실
- 정원/현재 인원 (10/15)
- 개강일~종강일
- **교습비 (수강료)**
- **기타청구 항목** (교재비, 재료비 등)
- 상태 (모집중/진행중/종료)
- 액션: [상세] [수강생관리] [수정] [삭제]

**수업 개설 폼** (모달)
- 수업명, 과목
- **담당 강사 배정** (드롭다운)
- 요일/시간 선택 (다중 요일 가능)
- 강의실, 정원
- 개강일, 종강일, 총 수업 횟수
- **교습비 설정**
  - 기본 수강료
  - 할인가 (조기등록, 형제할인 등)
- **기타청구 설정**
  - 교재비, 재료비, 기타 항목별 금액
- [저장] [취소]

**수업 상세** (클릭 시)
- 라우트: `/classes/detail?id={classId}`
- 탭 구성:
  - **수업정보**: 수정 가능, 교습비/기타청구 수정
  - **수강생 목록**: 추가/제외
  - **강사 배정**: 담당 강사 변경, 보조 강사 추가
  - **수업일정**: 캘린더, 휴강/보강 설정
  - **출결현황**: 수업별 출석률
  - **수납현황**: 수강료 납부 현황

### 2. 수업 신청
**라우트**: `/classes/enroll`

**주요 기능**
- STEP1: 학생 선택 (검색)
- STEP2: 수업 선택 (목록에서 선택)
- STEP3: 수강 정보 입력
  - 수강 시작일
  - 교습비 확인 (수업별 설정 자동 반영)
  - 할인 적용 (조기등록, 형제할인 등)
  - 기타청구 항목 선택
  - 결제 방법
- [신청완료] 버튼

### 3. 시간표 관리
**라우트**: `/classes/timetable`

**주요 기능**
- **전체 시간표 조회**
  - 주간/월간 뷰
  - 강의실별 시간표
  - 강사별 스케줄
  - 수업별 색상 구분

- **시간표 편집**

## 📌 메뉴 구성 핵심 원칙

### 1. 간결성
- **1depth**: 11개 (홈 포함)
- **2depth**: 평균 3개 (소통관리만 3개, 나머지 2-3개)
- **3depth**: 사용 안 함 ✅
- 상세 기능은 탭/모달로 처리

### 2. 직관성
- 메뉴명은 한글로 명확하게
- 아이콘으로 시각적 구분
- 자주 쓰는 기능은 빠른 메뉴에

### 3. 확장성
- 각 2depth에 기능 통합
- 추후 확장은 탭 추가로 대응
- 필요시 2depth 추가 (최대 4개까지)

### 4. 실용성
- 학원 현장에서 실제로 필요한 기능 중심
- 복잡한 기능은 Phase 3-4로 후순위

### 5. 확장성 (중대형 학원 지원)
- 소형 학원부터 중대형 학원까지 대응
- 다지점(캠퍼스) 운영 지원
- 지점별 데이터 관리 및 통합 분석
- 고급 기능은 선택적 활성화
- 학원 규모에 따른 단계별 기능 확장
